# Sign-Up Form
A sign up form built from scratch using HTML and CSS. 

This project is part of [The Odin Project's Full Stack Javascript path](https://www.theodinproject.com/paths/full-stack-javascript). It applies intermediate HTML and CSS concepts such as accessibility, typography considerations, advanced selectors, and form structure and validation.

## Installation
1. Clone the repository
2. Navigate to the project folder: ```cd sign-up-form```
3. Open index.html in your browser
